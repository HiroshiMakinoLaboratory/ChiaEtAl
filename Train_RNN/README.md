# FORCE RNN

#### Training Models

1. Clone
```
git clone https://github.com/HiroshiMakinoLaboratory/ChiaEtAl2023NatureCommunications.git
```

2. Create a conda environment
```
cd ChiaEtAl2023NatureCommunications/Train_RNN
conda env create -f environment.yml
conda activate rnn
```

3. Training
```
python training.py
```

4. Distract
```
python distract.py
```
